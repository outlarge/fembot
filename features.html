Функционал

Бот предназначен для поиска и анализа информации, основанной исключительно на общедоступных сообщениях Discord.

Команды бота

.search <user_id>
Поиск сообщений, опубликованных пользователем с указанным техническим идентификатором Discord в открытых серверах и каналах.
Публичные источники
.search <username>
Поиск сообщений по имени пользователя или никнейму, если он присутствует в общедоступном контенте.
Агрегированные данные
.profile <user_id | username>
Отображает обобщённую статистику активности пользователя на основе открытых сообщений без идентификации личности.
Без персональных данных
.support
Позволяет связаться с поддержкой по поводу технических вопросов, или идей по улучшению системы.
